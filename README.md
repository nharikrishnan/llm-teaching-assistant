# Teaching Assistant

AI-powered learning platform that turns PDF documents into interactive courses with video lessons, quizzes, chat, and real-time industry insights.

## Features

- **PDF Upload** -- Upload any PDF and get it automatically split into structured topics.
- **AI Lessons** -- Each topic gets a clear, structured explanation generated by an LLM.
- **Video Lessons** -- Auto-generated narrated slide videos (OpenAI TTS + Pillow + MoviePy).
- **Interactive Quizzes** -- AI-generated questions with LLM-based answer evaluation.
- **Chat with Teacher** -- Ask questions grounded in the document content.
- **Industry Standards** -- Real-time web search (DuckDuckGo) with LLM-synthesized summaries.

## Architecture

- **Frontend**: Next.js 15 + TypeScript + Tailwind CSS
- **Backend**: FastAPI + LangChain + OpenAI/Anthropic

## Quick Start

### Prerequisites

- Python 3.10+
- Node.js 18+
- OpenAI API key (required for LLM and TTS)

### Backend

```bash
cd backend
pip install -r requirements.txt
cp .env.example .env
# Edit .env and add your API key
uvicorn backend.main:app --reload
```

The API runs at `http://localhost:8000`.

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The app runs at `http://localhost:3000`. API requests are proxied to the backend automatically.

### Environment Variables

Create `backend/.env` from `backend/.env.example`:

| Variable | Required | Description |
|----------|----------|-------------|
| `DEFAULT_PROVIDER` | Yes | `openai` or `anthropic` |
| `OPENAI_API_KEY` | Yes (if openai) | OpenAI API key (also needed for TTS video generation) |
| `ANTHROPIC_API_KEY` | Yes (if anthropic) | Anthropic API key |
| `OPENAI_MODEL` | No | Override default OpenAI model (default: gpt-4o) |
| `ANTHROPIC_MODEL` | No | Override default Anthropic model |
